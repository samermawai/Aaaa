Help & FAQ

Bot Commands

Command Description
/start Welcome message and bot instructions.
/connect Start searching for a chat partner.
/disconnect End the current chat conversation.
/reveal Request to reveal identities (triggers inline buttons).
/invite Generate an invite link for your group.
/broadcast <message> (Admin-only) Send a message to all users.

Your privacy is our top priority. Messages are forwarded anonymously, and no personal information (names, usernames, or IDs) is shared between users unless both explicitly agree through the /reveal command. All conversations are private between matched users only.

When you use /connect, you're added to a waiting pool. The bot then randomly pairs you with another waiting user. If no partner is available immediately, you'll wait until someone else joins. After 45 seconds of waiting, you'll receive a timeout message with a button to try connecting again.

When you use /disconnect, the conversation ends immediately for both users. Your partner will receive a notification that you've disconnected. All pending identity reveal requests are automatically canceled, and you're free to start a new chat with /connect.

Either user can initiate an identity reveal request with /reveal. The partner then receives a message with Yes/No buttons. If they click Yes, both users receive each other's names and usernames (if available). If they click No, the request is declined, and the requester is notified. Identity reveals require mutual consent - both users must agree.

If you encounter inappropriate behavior, use /disconnect immediately to end the conversation. You can report issues to the bot admin using regular Telegram features. Remember to always be respectful and follow Telegram's Terms of Service when using the bot.
